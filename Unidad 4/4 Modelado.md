
En esta fase se construyen y evalúan modelos de aprendizaje automático para resolver el problema definido en la fase de comprensión del negocio.

## División de datos

Es el proceso de dividir el conjunto de datos en dos grupos:
* Grupo de entrenamiento: Se utiliza para construir el modelo.
* Grupo de pruebas: Se utiliza para evaluar el rendimiento del modelo.

La división de datos es una tarea importante para garantizar que el modelo sea generalizable.

Debe ser realizada de forma aleatoria para evitar sesgos.

## Construcción de modelos

Es el proceso de elegir los algoritmos de aprendizaje automático que se utilizarán para construir el modelo. Hay muchos algoritmos de modelado disponibles, cada uno con sus propias fortalezas y debilidades.

La selección de algoritmos de modelado debe basarse en los objetivos del proyecto, los datos disponibles y las limitaciones del sistema.

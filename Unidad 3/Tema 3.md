
## Introducción a los Algoritmos de Aprendizaje Automático Supervisado

### Naive Bayes

Es un algoritmo de aprendizaje automático probabilístico que se utiliza para la clasificación. Naive Bayes asume que las variables independientes son independientes entre sí.

Esto significa que la probabilidad de que un dato pertenezca a una categoría no se ve afectada por el valor de otras variables independientes.

### Regresión lineal

Es un algoritmo de aprendizaje automático que se utiliza para predecir un valor continuo a partir de un conjunto de datos.

Utiliza una línea para aproximar la relación entre las variables independientes y la variable dependiente. 

Se utiliza en una amplia gama de aplicaciones, como la predicción de ventas, la predicción de precios y la predicción de riesgos.

### Regresión logística

Es un algoritmo de aprendizaje automático que se utiliza para predecir la probabilidad de que ocurra un evento.

Utiliza una función logística para aproximar la relación entre las variables independientes y la variables dependiente, es ideal para la clasificación.

### Máquinas de vectores de soporte (SVM)

Identifican los puntos de datos que están más cerca de las fronteras entre las clases, el algoritmo también se adapta a problemas de regresión.

Las SVM identifican los puntos de datos que están más cerca de las fronteras entre las clases. Estos puntos de datos se denominan vectores de soporte.

### Árboles de decisión

Es un algoritmo de conjunto que combina múltiples árboles de decisión. Se divide al población o muestra en conjuntos basados en la variables de entrada más significativa.

### Redes neuronales

Son un tipo de algoritmo de aprendizaje automático que se inspira en el funcionamiento del cerebro humano.

Las redes neuronales se componen de una serie de nodos conectados entre sí. Cada nodo representa una función matemática que se utiliza para procesar información.

### LDA - Linear discriminant analisis

Es un algoritmo de reducción de dimensionalidad que se utiliza para separar dos o más clases de datos.

Calcula las direcciones principales de varianza entre las clases y, luego, proyecta los datos en estas direcciones.
